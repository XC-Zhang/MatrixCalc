Matrixcalc

A matrix calculator on android device.

View project onGitHub

Download APK

Requirement

Minimum Android version : 4.0

Usage

After launching the app, add several matrices by touching "Add" button or in action bar.


Enter rows, columns and name of matrix, then fill in the elements. Touch "OK" when finished.

Repeat these steps until you have added enough matrices.

Press buttons and operators to complete the expression. Then press "=" button.


You can edit or delete a matrix by long-pressing the button with the name of the matrix.
